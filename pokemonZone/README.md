# Pokemon Zone
Brief description of the application's functionality, eg., An interactive application for managing pokemon data persistence.

## Objective
This repository was created with the primary goal of demonstrating a solid command of integrating and manipulating data from external sources, as well as managing clientside data persistence.

The two key technological pillars of this project are:

### 1. External API Integration
- Skill Demonstrated: Efficient consumption and handling of JSON data retrieved from an external API.
- Implementation: The Pokemon API is used to fetch real-time information, including the characteristic, and image of each pokemon.
- Techniques Used: Implementation includes asynchronous calls (fetch) and handling loading states and errors.

### 2. Data Persistence with Local Storage
- Skill Demonstrated: Effective use of localStorage to store and retrieve information directly withinm the user's browser.
- Implementation: This feature allows the user to save their favorite pokemons between sessions.
- Benefit: Enhances the user experience by providing seamless persistence without requiring a server-side database.

## Technology Used
- Language: JavaScript
- Frontend: HTML5, CSS3
- Data Handling: fetch
- Storage: localStorage

## Installation and Setup
Follow these steps to run the project in your local environment:

1. Clone the repository: git clone https://github.com/vargasc475/Portfolio
2. Navigate to the project directory: cd pokemonZone
3. Open the index.html file in your browser
